# WooCommerce Custom Design (Riverr Shop)

This project includes custom CSS styling for a WooCommerce-based e-commerce website.

## Features
- Responsive design (mobile / tablet / desktop)
- Custom header and navigation
- Product page optimization
- Cart and checkout UI improvements
- Floating buttons (WhatsApp, Instagram)
- Clean modern UI/UX

## Technologies
- CSS3
- WordPress (WooCommerce)

## Goal
To improve user experience and create a modern, premium e-commerce interface.
